# 工作与私人生活繁荣的统计分析

## 📋 项目简介

本项目通过多元统计分析方法，探索工作与私人生活之间的关系，分析它们对个人生活幸福指数和成功的影响。该研究基于大规模数据集，运用多种统计学和数据挖掘技术来揭示工作与生活平衡的规律。

## 🎯 研究目标

- 分析工作投入与私人生活质量的关联性
- 探索影响生活繁荣度的关键因素
- 提供基于数据的生活平衡建议
- 验证不同职业和生活方式对整体幸福感的影响

## 📈 主要发现
- 工作时长与生活满意度呈现非线性关系
- 私人生活投入对整体幸福感的影响显著
- 不同职业类别的工作-生活平衡差异明显

## 📝 关键指标

| 指标 | 描述

| 工作-生活平衡指数 | 衡量工作与生活协调程度 |
| 生活繁荣度 | 综合反映个人生活质量 |
| 幸福指数 | 主观生活满意度评分 |

## 🔍 统计方法

- **描述性统计**：均值、标准差、分布分析
- **推断统计**：假设检验、置信区间
- **相关分析**：Pearson相关、Spearman秩相关
- **回归分析**：线性回归、多元回归
